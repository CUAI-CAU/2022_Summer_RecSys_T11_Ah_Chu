## Abstract
Amazon Review Dataset을 이용하여 추천시스템 모델을 구현한다. 그 과정에서 자연어처리 기술을 이용하여 Meta Data의 상품 description 항목을 임베딩하여 성능을 높였다. 또한 최신 모델인 BiVAE를 이용해 성능을 높일 수 있었다.
